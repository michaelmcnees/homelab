# `host-monitoring` role

Installs the Beszel agent and Prometheus `node_exporter` on a non-cluster host
(workstation, laptop, Mac Studio, kids' future PCs). Each tool runs as its
native daemon — Beszel agent for the Beszel hub UI, node-exporter scraped by
Prometheus over Tailscale.

Tailscale itself is installed and joined out-of-band (different lifecycle).

## Supported platforms

- **macOS** (Apple Silicon and Intel) — agents installed via Homebrew
- **Linux** (Debian/Ubuntu/Fedora/Arch on amd64 or arm64) — agents installed
  via package managers + a systemd unit when needed

## What it installs

| Component | macOS | Linux |
|---|---|---|
| Beszel agent | `brew install beszel/beszel/beszel-agent` + `brew services start` | `apt`/`dnf`/`pacman` if available, else release tarball + systemd unit |
| node-exporter | `brew install node_exporter` + `brew services start` | distro package + systemd unit; binds to `0.0.0.0:9100` |

## Variables

| Variable | Default | What |
|---|---|---|
| `host_monitoring_beszel_hub_url` | `http://beszel.tailnet.example.ts.net:8090` | Beszel hub URL the agent connects to (over Tailscale or LAN) |
| `host_monitoring_beszel_agent_key` | _(required)_ | Public key generated by the Beszel hub when adding the system. Per-host. |
| `host_monitoring_node_exporter_listen` | `0.0.0.0:9100` | node_exporter listen address. Tailscale-only hosts can scope to the tailnet IP. |
| `host_monitoring_node_exporter_collectors_extra` | `[]` | Extra `--collector.*` flags |

## Inventory pattern

Add a `household_hosts` group in `ansible/inventory/hosts.yml`:

```yaml
household_hosts:
  hosts:
    michael-mbp:
      ansible_host: michael-mbp.tailnet.example.ts.net
      ansible_user: michael
      host_monitoring_beszel_agent_key: REPLACE_ME_PER_HOST
    mac-studio:
      ansible_host: mac-studio.tailnet.example.ts.net
      ansible_user: michael
      host_monitoring_beszel_agent_key: REPLACE_ME_PER_HOST
  vars:
    host_monitoring_beszel_hub_url: http://beszel.tailnet.example.ts.net:8090
```

Then run:

```sh
ansible-playbook -i inventory/hosts.yml playbooks/host-monitoring.yml
```

## After install

1. Open the Beszel hub UI, confirm the new host appears as connected.
2. In Prometheus, confirm the new `node_exporter` target is `up=1`. Static
   targets are configured in
   [`kubernetes/infrastructure/observability/kube-prometheus-stack/household-hosts-scrape.yaml`](../../kubernetes/infrastructure/observability/kube-prometheus-stack/household-hosts-scrape.yaml).
